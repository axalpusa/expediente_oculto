# Contexto Inicial - Expediente Oculto

## Descripción
"Expediente Oculto" es un proyecto de gamificación diseñado para educar sobre el periodismo de investigación, el acceso a la información pública (Ley N° 27806) y la fiscalización ciudadana. El juego sumerge al usuario en el papel de un reportero que investiga presuntas irregularidades en la reconstrucción del Teatro Municipal de Piura.

## Propósito Educativo
El objetivo principal es enseñar a los usuarios a:
- Contrastar rumores de redes sociales con documentos oficiales (CUI, expedientes técnicos).
- Identificar irregularidades comunes en obras públicas (invasión de terreno, falta de supervisión, retrasos).
- Entender el valor de la transparencia y el control concurrente de la Contraloría.

## Palabras Clave
Gamificación, Transparencia Pública, Periodismo de Investigación, Piura, Teatro Municipal, CUI 2523322.

## Contenido Técnico Inicial
- **Tecnologías:** HTML5, CSS3 (Vanilla), JavaScript (Vanilla).
- **Arquitectura:** Aplicación de una sola página (SPA) basada en estados de visibilidad de secciones (`.screen`).
- **Mecánicas:**
  - Simulación de interfaz de "post" de noticias.
  - Arrastre y soltado (Drag & Drop) para fact-checking.
  - Desbloqueo mediante códigos numéricos (CUI).
  - Exploración de escenas interactivas.

## Estado de Seguridad (Junio 2026)
El proyecto fue sometido a una refactorización de seguridad para eliminar elementos que imitaban visualmente a redes sociales comerciales (Twitter/X), mitigando así bloqueos por "sitio peligroso" en plataformas de hosting como GitHub Pages.
